Política de Privacidade

Última atualização: 08 de fevereiro de 2026

A sua privacidade é importante para nós. Esta Política de Privacidade descreve como o aplicativo ConectaFé coleta, usa e protege suas informações.

1. Coleta de Informações

O ConectaFé foi projetado para ser o mais privado possível. Não exigimos criação de conta para leitura de orações. Coletamos apenas:

  • Dados de Uso: Informações anônimas sobre quais funcionalidades são mais utilizadas para melhoria do app.
  • Dados de Pagamento: Caso você realize uma doação, as informações financeiras (cartão de crédito) são processadas diretamente pelo Stripe.

2. Processamento de Pagamentos (Stripe)

Para doações, utilizamos o processador de pagamentos Stripe. O ConectaFé não armazena seus dados de cartão de crédito em nossos servidores. Todas as transações são criptografadas e processadas de acordo com os padrões de segurança PCI-DSS estabelecidos pelo Stripe.

3. Arquivos Locais e JSON

O aplicativo utiliza arquivos locais (JSON) para exibir orações. Não acessamos sua galeria de fotos, contatos ou localização, a menos que explicitamente solicitado por uma funcionalidade futura e autorizado por você.

4. Compartilhamento de Dados

Não vendemos, trocamos ou transferimos suas informações para terceiros. Os únicos dados compartilhados são os necessários para o processamento de pagamentos com o Stripe.

5. Segurança

Implementamos medidas de segurança para manter a segurança de suas informações pessoais quando você realiza uma doação ou acessa o aplicativo.

6. Alterações na Política

Podemos atualizar nossa Política de Privacidade periodicamente. Recomendamos que você revise esta página para quaisquer mudanças.
